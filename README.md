# Renegociação API

Projeto desenvolvido com foco em DevOps, CI/CD e TDD.

## Tecnologias
- .NET 8
- NUnit
- GitHub Actions
- Docker

## Pipeline
A cada commit:
- Build automático
- Testes automatizados
- Geração de artefato

## Testes
Cobertura com testes unitários garantindo regras de negócio da renegociação.

## Execução
dotnet run
